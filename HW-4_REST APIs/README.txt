1. Download 'Postman'
2. Run the app.py
3. Paste url to Postman
4. Go to 'header' and set 'KEY' to 'Content-Type', AND 'VALUE' to 'application/json'
5. Go to 'body' and chose 'raw', and send some raw JSON
6. Set the appropriate method (ex: post, get, etc...) then hit send
